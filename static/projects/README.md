# Project Screenshots

Folder ini untuk menyimpan screenshot dari project-project portfolio.

## Cara Menggunakan

1. Simpan screenshot project Anda di folder ini
2. Format nama file yang disarankan:
   - `shopsmart.png` - Screenshot ShopSmart E-Commerce
   - `ai-support.png` - Screenshot AI Customer Support System
   - `portfolio-rag.png` - Screenshot Portfolio RAG Website
   - `taskflow.png` - Screenshot TaskFlow App

3. Update URL gambar di `templates/index.html`:
   - Ganti URL Unsplash dengan: `/static/projects/nama-file.png`
   - Contoh: `'/static/projects/shopsmart.png'`
